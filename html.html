HTML is the most vital language used for making websites, as every website uses HTML in some way, shape, or form. However, HTML just by itself is not as useful as it may seem, and is only able to create a "skeleton" of a webpage. It needs other languages (like CSS and bootstrap) to make things look more appealing and altogether better designed. The easiest way to look at the subject is think of HTML as the language used to put information on a page, and then is enhanced via css/bootstrap to seem more appealing, become more useful, or display more information.

    *all commands have a "<" before the command and a ">" after the command, in order to keep the code more organized.

  • !DOCTYPE ...: defines what type of document your writing in
  • head: defines the head of your website nothing will show up here so it's mainly used for putting setup commands in
  • body: defines the body of the document
  • button: used to make buttons
  • a: used to link things to a button
  • img: used to add images
