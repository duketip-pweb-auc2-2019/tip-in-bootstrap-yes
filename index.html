Programming for the Web

Programming for the Web is a class at the Duke TIP Program where students learn to create their own webpages from scratch.

Programming for the Web (or PWeb, for short) utilizes programming languages such as HTML, CSS, and Bootstrap to create Webpages both for function and for style.

About Duke TIP (Talent Identification Program)

Duke TIP is a program in which students with acedemic talent are selected to participate in a three week long summer program. The students selected are allowed to choose a particular subject to take; Among some of the variety of classes are; Crime Scene Investigation, Architecture, and most important for our purposes, Programming for the Web.



The Duke TIP program is a program that searches for students that are considered "acedemically talented", and invites them to enter their program. However, to qualify, they must take a test: the SAT/ACT. They must score an above average score to qualify. Students in Duke TIP score about 18(ACT) and 1000(SAT) on average). Once qualified, the students must choose their desired class; for the students of PWeb, they chose, PWeb as their course. The program takes place during the summer, where students spend three weeks exploring the class they have chosen.


About Programming for the Web

In the class, "Programming for the Web", students learn to create entire webpages from near scratch, with a variety of programming languages, many of which intersect into each other to create websites with a variety of complexities. The major language used for creating these webpages is basic HTML, which is then enhanced by languages such as CSS and Bootstrap, the latter of which is used widely by professionals to enchance their more complex projects.

About HTML

HTML is, in essence, the "framework" of all websites. Every website on the internet, even if made with a "website builder", such as wix, uses some form of HTML in order to function.

Despite it's importance in website creation, HTML is very limited on it's own. It requires other programming tools, such as CSS and Bootstrap, to appear more appealing, and to have a stronger function.

About CSS

CSS is one of the simplest tools used to add a higher level of quality to an HTML webpage. It lets you do things like change the color, change the font and size of text and lets you format it how you like. HTML is, in general, always paired with CSS, along with whatever other HTML "enhancer" is being used for that specific program.

About Bootstrap

Bootstrap essentialy is a simplified form of CSS that has the ability to add much, much more, due to the fact that it adds classes and presets that allow for you to write less when formatting text and changing your page. However, Bootstrap does add a number of more complicated commands, although they can add a much higher level of quality and depth to an sHTML/CSS website.

HTML, CSS, and Bootstrap.

When working together well, HTML, CSS, and Bootstrap can be used to create websites of professional quality, although Bootstrap is by no means the only HTML enhancer. To conclude, HTML writes the webpage, CSS outfits the webpage, and Bootstrap enhances the webpage.
